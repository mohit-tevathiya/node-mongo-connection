This repo accomadates connecting node js with mongo db without any ORM's for mongo

Topics to be covered : 
1)Error handling of database if database does not exist/invalid or the collection does not
exist/invalid
2)Create modular code .Database file will accomadated in different file.

Some stuff which will app is not capable to handle at the mommet:

1)Error handling for inexistent database or collection.


Steps to install :
1)Clone the repo https://github.com/mohit-tevathiya/node-mongo-connection.git
2)cd mongodb-to-node-conection
3)Run npm install to install all the dependencies
4)Run nodemon index.js
